# Assets Directory

This directory is for image assets referenced in the README.

## Files

- `wechat-qr.png` - WeChat group QR code (replace this file with your actual QR code image)

## Adding the WeChat QR Code

1. Generate your WeChat group QR code (WeChat > Group Chat > ... > Group QR Code)
2. Save the image as `wechat-qr.png` in this directory
3. The README.md will automatically display it

## Recommended Image Specs

- Format: PNG
- Size: 200x200 to 400x400 pixels
- File size: < 100KB for fast loading
