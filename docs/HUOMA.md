# 群活码生成指南

## 什么是群活码?

普通的微信群二维码:
- 群满 100 人就扫不进去了
- 二维码图片换了,所有引用地方都要换

**群活码:**
- 一个固定的 QR 图片
- 扫进去是一个**中间网页**(类似草料二维码生成的)
- 网页里放多个群的二维码
- 一个群满了,在活码后台**改指向**到下一个群
- **用户扫的图永远不变**,不需要重新发图

## 5 分钟生成步骤(以草料为例)

### Step 1: 登录草料

打开 https://cli.im/ → 用微信扫码登录

### Step 2: 创建活码

- 左侧菜单 → "活码"
- 点 "新建活码"
- 填名称:"free-ai-router 交流群"

### Step 3: 配置跳转

- 上传多个微信群二维码图片(放满 3-5 个备用群)
- 设置"二维码切换规则":
 - 一个群 7 天没人扫就停用
 - 满 80 人就切换下一个
- 生成活码图片

### Step 4: 下载 + 放到项目

1. 下载生成的活码图片
2. 保存到 `assets/wechat-qr.png`(直接覆盖占位)
3. git add + commit + push
4. README 里的图片会自动显示

## 注意事项

1. **每个微信群最多 100 人**(免费版),满了再建新群
2. **定期检查活码链接**,确保跳转正常
3. **备用 3-5 个群**,避免主群满员无人可加
4. **不要在活码后挂广告**,容易被举报封号

## 推荐布局(活码后)

活码扫进去的页面可以放:
- 项目 GitHub 链接
- 项目介绍
- 群规
- 备用群(满了自动跳)
- 公众号(可选)
